# M-mut.7 — static mutable + set-hooks (mutation milestone, final slice)

> Slice 7 of the mutation milestone (`docs/specs/2026-06-21-mutation-milestone-design.md` §4) — the
> last slice: program-lifetime `static mutable` class fields, plus property **set-hooks**.
> Autonomy: directive `f5f2912`.

## Decisions Log
- [2026-06-21] AGREED: **static field access = dot `Counter.total`** (NOT `::`). Forced-consistent with
  Phorge's `.`-everywhere decision (Go-qualified calls, bare enum variants, no `::` token). The head is
  a class name (never a value) ⇒ unambiguous vs instance member access. PHP target: `Counter::$total`.
- [2026-06-21] AGREED: **M-mut.7 scope = build BOTH static mutable AND set-hooks now** (developer chose
  "both" over the 7a/7b split). Set-hooks get a design pass first, then build.
- [2026-06-21] NEW FEATURE REQUESTED (orthogonal to mutation): **class-level `public`/`private`
  visibility** — classes default `public`; `private class` = **not usable outside its `.phg` file**
  (Java file-scoping). Its own design pass needed (the M5 loader flattens all package files into one
  AST, so file provenance must be tracked; PHP has no file-private classes ⇒ checker-enforced only,
  transpiles to a normal PHP class — byte-identity holds because Phorge never emits a cross-file ref).
  Sequenced AFTER M-mut.7 unless the developer redirects.

## Build order
1. **7a — static mutable** (`static mutable int total = 0;`, `Counter.total` read/write). New
   `Op::GetStatic(idx)` / `Op::SetStatic(idx)`; `Modifier::Static`; field-level initializer
   (`ClassMember::Field.init: Option<Expr>`, literal-const only this slice). Static store: interpreter
   `HashMap<(class,field), Value>`; VM a `statics: Vec<Value>` seeded from a compiled init table at
   start. Checker: `Counter.total` resolves the head class → static field; `E-STATIC-*` codes.
   Initializer timing = once at program load (PHP const-expr model, §8.4). TDD → byte-identical → commit.
2. **7b — set-hooks** (design then build; likely pairs with the deferred get-hooks as one PHP
   property-hook feature). Design pass first.
