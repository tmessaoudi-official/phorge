# M3 Slice-1 · S0 (Developer Experience) Implementation Plan

> **For agentic workers:** Execute task-by-task. Steps use checkbox (`- [ ]`) syntax. Each **Task**
> is one green commit. Execution is **inline** (not subagent-driven): S0.2–S0.4 all touch the same
> files (`ast.rs`, `parser.rs`, `checker.rs`, `transpile.rs`, `cli.rs`), so parallel subagents would
> collide — inline + sequential is the only safe mode.

**Goal:** Land slice S0 (developer experience) from the M3 roadmap — per-command `--help` with
examples, `var` local type inference, `type` aliases, and sharper diagnostics (caret + hint) plus a
`phorge explain <code>` subcommand — with **zero backend/`Op` change** and the `run`≡`runvm`
byte-identity spine intact.

**Architecture:** Four independent DX features. `var` adds `Type::Infer` (resolved in the checker,
erased everywhere else). `type` adds `Item::TypeAlias` (registered in `collect`, expanded in
`resolve_type`, erased in every backend). Diagnostics gain `code`/`hint` fields + a source-aware
caret renderer used only for **front-end** (lex/parse/type) errors — runtime error strings are left
byte-identical so the differential `agree_err` oracle is unaffected. `explain` is a standalone
code→prose table.

**Tech Stack:** Rust 2021, std-only, `#![forbid(unsafe_code)]`, `[lints] warnings = "deny"` (so a
non-exhaustive `match` is a hard compile error — the safety net for the `Item`/`Type` coupling).

**Gate (run before every commit):** `export PATH=/stack/tools/cargo/bin:$PATH` then
`cargo test && cargo clippy --all-targets && cargo fmt --check`.

```rust
    fn parse_var_inferred(&mut self) -> Result<Stmt, Diagnostic> {
        let sp = self.peek_span();
        self.expect(&TokenKind::Var, "'var'")?;
        let name = self.expect_ident("a variable name after 'var'")?;
        self.expect(&TokenKind::Eq, "'=' after 'var <name>'")?;
        let init = self.parse_expr()?;
        self.expect(&TokenKind::Semicolon, "';' after variable declaration")?;
        Ok(Stmt::VarDecl {
            ty: Type::Infer(sp),
            name,
            init,
            span: sp,
        })
    }
```

```rust
            Stmt::VarDecl {
                ty,
                name,
                init,
                span,
            } => {
                let actual = self.check_expr(init);
                let declared = match ty {
                    crate::ast::Type::Infer(_) => {
                        // `var`: the binding takes the init's type. If the init itself failed to
                        // check (Ty::Error — e.g. `var x = null;`, rejected at the init), propagate
                        // Error without emitting a second diagnostic.
                        actual.clone()

                    _ => {
                        let declared = self.resolve_type(ty);
                        if !Ty::assignable(&actual, &declared) {
                            self.err(*span, format!("expected `{declared}`, found `{actual}`"));
                        }
                        declared
                    }
                };
                self.declare(name, declared);
            }
```

```rust
            if let Some(line_text) = src.lines().nth((self.line - 1) as usize) {
                s.push('\n');
                s.push_str(line_text);
                if self.col > 0 {
                    s.push('\n');
                    // Indent the caret by the column, preserving tabs so it lines up in the
                    // terminal regardless of tab width.
                    let pad: String = line_text
                        .chars()
                        .take((self.col - 1) as usize)
                        .map(|c| if c == '\t' { '\t' } else { ' ' })
                        .collect();
                    s.push_str(&pad);
                    s.push('^');
                }
            }
        }
        if let Some(code) = self.code {
            s.push_str(&format!("\n  [{code}]"));
        }
        if let Some(hint) = &self.hint {
            s.push_str(&format!("\n  hint: {hint}"));
        }
        s
    }
```

```rust
/// Classic two-row Levenshtein edit distance (ASCII-oriented; identifiers are ASCII in M1).
fn levenshtein(a: &str, b: &str) -> usize {
    let (a, b): (Vec<char>, Vec<char>) = (a.chars().collect(), b.chars().collect());
    let mut prev: Vec<usize> = (0..=b.len()).collect();
    let mut cur = vec![0usize; b.len() + 1];
    for (i, ca) in a.iter().enumerate() {
        cur[0] = i + 1;
        for (j, cb) in b.iter().enumerate() {
            let cost = usize::from(ca != cb);
            cur[j + 1] = (prev[j + 1] + 1).min(cur[j] + 1).min(prev[j] + cost);
        }
        std::mem::swap(&mut prev, &mut cur);
    }
    prev[b.len()]
}
```

```rust
            Phorge resolves identifiers lexically: locals (including `var` bindings and `for` loop\n\
```
